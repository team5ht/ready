# Syllable Trainer — split stage 2

Второй этап разрезания без намеренного изменения поведения.

Структура:

```text
index.html
css/styles.css
js/config.js
js/audio.js
js/game.js
js/drag.js
js/main.js
audio/
```

Файлы аудио должны лежать в `audio/` рядом с `index.html`, например:

```text
audio/make_syllable.mp3
audio/need.mp3
audio/success.mp3
audio/ma.mp3
...
```

Порядок подключения JS-файлов в `index.html` важен.
